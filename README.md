# birdir1 Ops Panel

Central operations console for the entire birdir1 portfolio.

## Purpose
- Portfolio status, checklist progress, reports and assets in one place.
- Not tied to a single product admin panel.

## Run (local)
```bash
npm install
npm run dev
```

## Environment
- `OPS_ADMIN_PASSWORD` (required) — login password for the ops panel.

## Data Sources
- `data/portfolio-apps.json`
- `data/inventory.json`
- `data/checklist.md`
- `data/reports/`
- `data/uploads/`
